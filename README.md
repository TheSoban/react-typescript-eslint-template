## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

```sh
git clone https://github.com/TheSoban/react-typescript-eslint-template
cd react-typescript-eslint-template
yarn install
```

### Running the application

```sh
yarn dev
```
